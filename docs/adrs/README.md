# ADRs

Architecture Decision Records capture durable technical and operating choices that shape the prototype.

## Rules

- Use a new numbered ADR for each significant design decision.
- Keep ADRs immutable after acceptance except for status changes or clarifying notes.
- Link ADRs from the mission tracker, decision log, invariants, or evidence manifest when relevant.
