# Examples

This directory contains reproducible example inputs and executable example scenarios.

Current contents:

- `examples/policies/` for validating `CPL v0.1` policy packages
- `examples/inventory/` for normalized candidate inventory sets consumed by `make policy-eval`
- `examples/obligations/` for normalized obligation inputs consumed by `make optimize`
- `examples/demo-scenarios/margin-call/` for the IDE-ledger and Quickstart-backed positive and negative end-to-end margin-call demo bundles
- `examples/demo-scenarios/return/` for the IDE-ledger and Quickstart-backed positive and negative end-to-end return demo bundles consumed by `make demo-return` and `make demo-return-quickstart`
- `examples/demo-scenarios/substitution/` for the positive and negative end-to-end substitution demo bundle consumed by `make demo-substitution`

Future additions should remain executable or schema-validating rather than illustrative-only.
