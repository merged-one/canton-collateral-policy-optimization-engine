# Reports Surface

This directory contains report schemas and generated outputs that come from real execution paths.

Rules:

- machine-readable reports belong here once their contracts exist
- generated outputs must be traceable to real commands
- placeholder or fabricated success artifacts do not belong here

Current contents:

- `reports/schemas/policy-evaluation-report.schema.json` for the first machine-readable policy decision artifact
- `reports/schemas/optimization-report.schema.json` for the first machine-readable optimization artifact
- `reports/schemas/execution-report.schema.json` for the first end-to-end workflow execution artifact
- `reports/schemas/return-report.schema.json` for the first end-to-end return execution artifact
- `reports/schemas/substitution-report.schema.json` for the first end-to-end substitution execution artifact
- `reports/generated/central-bank-domestic-window-policy-central-bank-eligible-set-policy-evaluation-report.json` generated by `make policy-eval`
- `reports/generated/central-bank-domestic-window-policy-central-bank-eligible-set-central-bank-window-call-optimization-report.json` generated by `make optimize`
- `reports/generated/margin-call-demo-execution-report.json`, `reports/generated/margin-call-demo-summary.md`, and `reports/generated/margin-call-demo-timeline.md` generated by `make demo-margin-call`
- `reports/generated/return-demo-report.json`, `reports/generated/return-demo-summary.md`, and `reports/generated/return-demo-timeline.md` generated by `make demo-return`
- `reports/generated/substitution-demo-report.json`, `reports/generated/substitution-demo-summary.md`, and `reports/generated/substitution-demo-timeline.md` generated by `make demo-substitution`
- `reports/generated/conformance-suite-report.json` and `reports/generated/conformance-suite-summary.md` generated by `make test-conformance`
- `reports/generated/final-demo-pack.json` and `reports/generated/final-demo-pack-summary.md` generated by `make demo-all`
